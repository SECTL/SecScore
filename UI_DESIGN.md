# SecScore UI 设计文档

本文档描述了 SecScore 应用程序的界面结构。您可以直接修改本文档中的描述，保存后发回给我，我将根据您的修改更新代码。

## 1. 侧边栏 (Sidebar)
- **头部**: 应用标题 "SecScore"
- **导航菜单**:
  - 学生管理 (StudentsPage)
  - 事件记录 (EventsPage)
  - 排行榜 (LeaderboardPage)
  - 设置 (SettingsPage)
- **底部**: 
  - **同步状态卡片** (SyncStatusCard): 显示当前同步状态（成功/失败）和最后同步时间。

## 2. 页面设计
注意：所有交互元素（按钮、输入框、下拉菜单等）都应居中对齐，且宽度不得超过所在对话框的80%宽度。
### 2.1 学生管理页 (StudentsPage)
- **布局**: 列表视图
- **列表项**:
  - 学生姓名 (居中显示)
  - 操作按钮组 (居中显示): [编辑] [删除]
- **底部**:
  - [添加学生] 按钮 (居中)

### 2.2 事件记录页 (EventsPage)
- **顶部**: 
  - 标题 "积分管理"
- **操作面板** (内嵌于标题下方):
  - 标题 "快速评分"
  - 输入项:
    - 学生 (下拉选择)
    - 理由 (下拉选择)
    - 分值 (数字输入)
    - 备注 (文本输入, 可选)
  - [提交评分] 按钮
- **内容区域**:
  - 标题 "最近记录"
  - 列表显示最近的积分操作历史

### 2.3 排行榜页 (LeaderboardPage)
- **顶部**: 
  - 标题 "排行榜"
  - 排序选项 (本周/本月/全部)
- **列表**:
  - 排名 | 学生姓名 | 总积分 | 趋势图标

### 2.4 设置页 (SettingsPage)
- **外观**:
  - 主题切换 (下拉选择)
- **运行模式**:
  - [本地模式] / [远程模式] 单选
- **网络设置** (仅远程模式显示):
  - WebSocket URL 输入框
  - 连接状态显示
- **理由管理**:
  - 理由列表 (显示名称)
  - 操作: [编辑] [删除]
  - [添加理由] 按钮
- **关于**:
  - 显示软件版本和介绍信息 (读取 ABOUT.md)

## 3. 全局组件
- **SyncStatusCard**: 封装了同步状态显示的卡片组件，可在多处复用。
